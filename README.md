# 🚨 AI Disaster Response Commander

An AI-powered disaster response system built using **Python, Flask, JavaScript, Leaflet.js, and SQLite** that intelligently allocates emergency resources during disasters.

---

# 📌 Features

- 🌍 Interactive Live Map
- 🚑 Nearest Ambulance Allocation
- 🏥 Nearest Hospital Selection
- 🤖 AI Decision Engine
- ⚠️ Disaster Severity Prediction
- 🎯 AI Confidence Score
- 👨‍🚒 Rescue Team Allocation
- 📋 Incident History
- 📊 Dashboard Statistics
- 💾 SQLite Database
- 🌐 Flask REST API

---

# 🚀 How to Run

## Clone the repository

```
git clone <repository-url>
```

## Install dependencies

```
pip install flask
```

## Start the application

```
cd backend

python app.py
```
